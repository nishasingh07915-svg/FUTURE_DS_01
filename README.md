Superstore Sales Dashboard

This project contains the sales dashboard created for Task 1. Dashboard Includes:

Total Sales

Total Quantity

Total Orders

Total Profit

Sales by Sub-Category

Profit by Catergory and Segment

Sales by Religion

Sales by Segment

Discount by Product Name

Monthly Sales Trend

Profit by Product Name

Filters (Ship Mode, Region, City, State)


Tools Used

*Power BI

*GitHub

*Data Cleaning (Power Query)

Insights

Technology category has highest sales.

Sales increasing year by year.

Some states generate higher revenue.

Discounts vary widely across products.
